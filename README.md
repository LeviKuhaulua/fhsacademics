# FHS Academics

Purpose: To spread awareness to students at Farrington High School about the Advanced Placement (AP) program. 
There students can learn the benefits of AP and how it can help prepare them for college. Additionally, students
can also be informed about events regarding AP to learn more about the program or help support it. 

# Getting Started

Using [pipenv](https://pipenv.pypa.io/en/latest/) as the package manager for this project. To get started, please do the following: 

- `pip install --user pipenv` 
- Navigate to the project folder and run `pipenv shell` to create an environment in the project folder and load up your virtual environment.
- Finally, run `pipenv install --ignore-pipfile` to install all the packages in the `Pipfile.lock` file. 
- **Exit Virtual Environment**: type exit into the terminal

&#x2606; Important that you add the `--ignore-pipfile` flag so that it doesn't install latest packages that could break the Django Server 


&#x2606; Check out the website (CLI Reference) to understand what each command does. 


&#x2606; I also added the standard `requirements.txt` file so that you can see as to what packages are currently being used in this project.
